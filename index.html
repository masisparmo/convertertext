Masukkan Kunci API Gemini

Untuk menggunakan fitur AI, masukkan kunci API Anda. Kunci akan disimpan di browser Anda.

ATAU

Gunakan di Lingkungan Gemini (Tanpa API)

Cara Mendapatkan Kunci API Gemini

  1. Buka Google AI Studio di aistudio.google.com.
  2. Login dengan akun Google Anda.
  3. Klik tombol "Get API key" di pojok kiri atas.
  4. Klik "Create API key in new project".
  5. Salin (copy) kunci API yang muncul.
  6. Tempel (paste) kunci tersebut ke kolom input di halaman ini.
